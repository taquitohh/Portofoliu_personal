## Cum să faci screenshots

1. Rulează `npm run dev` în folderul `website/`
2. Deschide http://localhost:4321/Portofoliu_personal
3. Folosește `Win + Shift + S` pentru a captura secțiuni
4. Salvează imaginile aici
